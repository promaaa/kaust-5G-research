# Plan: SOTA Presentation - UABS and 5G Deployments

## Objective
Create an Obsidian Advanced Slides presentation and speaker notes document reviewing literature on Unmanned Aerial Base Stations (UABS) and 5G deployments, validating the project's x86 architecture direction.

## Deliverables

### 1. Obsidian Advanced Slides Markdown
- File: `sota-pres1.md` (or similar in current directory)
- Format: Slide separators `---`
- 6 slides covering:
  1. Introduction to SOTA
  2. "Gold Standard" Prototypes (x86 Dominance)
  3. Network Architecture: IAB and Positioning
  4. Edge Cases and Hardware Limitations
  5. Synthesis & Conclusion
  6. Visual: Literature Hardware Matrix (Mermaid.js quadrantChart)

### 2. Speaker Notes Markdown
- File: `sota-pres1-notes.md` (or similar)
- Detailed analytical notes for each slide
- References to papers in `/paper` directory

## Content Structure

### Slide 1: Introduction to the SOTA
- Goal: Identify optimal hardware/software architectures for UAV-mounted 5G base stations
- Context: Literature review of 7 papers with ratings 1-5/5

### Slide 2: "Gold Standard" Prototypes (x86 Dominance)
- SkyCell (5/5): Intel NUC 7i7DNKE Mini PC on DJI M600Pro
- SkyRAN (4/5): Mini-PC with i7 processor on DJI M600Pro
- Key insight: Processing demands require x86 Mini-PCs

### Slide 3: Network Architecture: IAB and Positioning
- IAB in 5G (4/5): CU/DU separation concept
- Flying Rebots (3/5): Drone positioning optimization, x86 processor

### Slide 4: Edge Cases and Hardware Limitations
- Jetson Nano OAI (2/5): Struggles with monolithic 5G RAN
- Jetson Nano + OAI (2/5): Edge video processing, not base station
- Wi-Fi backhaul dismissal (1/5): Validates cellular IAB approach

### Slide 5: Synthesis & Conclusion
- Literature validates x86 Mini-PC architecture
- Stable, high-performance UAV-BS for OAI processing

### Slide 6: Literature Hardware Matrix (Mermaid.js quadrantChart)
- X-Axis: "Embedded ARM (Jetson)" to "x86 Mini-PC (Intel NUC)"
- Y-Axis: "Low Relevance/Success" to "High Relevance/Success (5/5)"
- Papers plotted showing x86 cluster in top-right
